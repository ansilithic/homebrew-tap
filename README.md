# homebrew-tap

Homebrew tap for [Ansilithic](https://github.com/ansilithic) command-line tools.

## Usage

```sh
brew tap ansilithic/tap
```

Then install any tool:

```sh
brew install ansilithic/tap/ping
brew install ansilithic/tap/rm
brew install ansilithic/tap/rmdir
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `ping` | Enhanced ping with timestamps, color-coded output, and packet loss summary |
| `rm` | Safe file deletion — moves files to Trash instead of permanent deletion |
| `rmdir` | Safe directory removal — moves directories to Trash instead of permanent deletion |
